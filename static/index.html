Como Funciona a APP?

Esta APP em Flask funciona com rotas, ao adicionar "/" após o URL e uma palavra chave leva a essa rota.

Lista de Rotas :

  • /path

    • mostra o subpath em que está

  • /hello

    • Mostra uma mensagem de Hello World

  • /user

    • mostra o subpath user

  • /projects

    • Texto de Projects

  • /about

    • Texto de About

  • /login

    • o Texto muda dependendo do login

  • /post

    • Mostra o subpath INT
